# Security and privacy

Please do not open a public issue containing a device identifier, network
address, host key, private key, password, access token, raw diagnostic report,
or screenshot from a private account. If you spot one in this repository,
contact the maintainer privately and include only the path and a short
description of the problem.

Ferrink is experimental software for modified devices. Keep an independent
recovery route before trying boot or framebuffer tools.
